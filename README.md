## ✨ ویژگی‌ها | Features

### 🎯 شش حالت کامل | Six Complete Modes

| # | حالت | توضیحات |
|---|------|---------|
| 1️⃣ | **Basic** | ماشین حساب ساده با طراحی Glassmorphism |
| 2️⃣ | **Scientific** | توابع sin، cos، tan، log، ln، π، e، توان و رادیکال |
| 3️⃣ | **Programmer** | تبدیل مبناهای BIN، OCT، DEC، HEX |
| 4️⃣ | **Graphing** | رسم نمودار توابع ریاضی به صورت زنده |
| 5️⃣ | **Unit Converter** | تبدیل واحد طول، وزن و دما |
| 6️⃣ | **Date Calculator** | محاسبه اختلاف روزها بین دو تاریخ |

### 🎨 ویژگی‌های طراحی و UX

- 🇮🇷 **پشتیبانی دوزبانه** - سوئیچ آنی بین فارسی و انگلیسی
- 🔄 **راست‌چین خودکار** - RTL شدن کامل برنامه در حالت فارسی
- 📳 **Haptic Feedback** - لرزش لمسی در تمام دکمه‌ها
- 💾 **ذخیره تاریخچه** - تاریخچه محاسبات به صورت دائمی
- 🎭 **انیمیشن‌های زنده** - پس‌زمینه‌های متحرک با افکت Neon Glow
- 🎨 **فونت Vazirmatn** - نمایش زیبای متون فارسی
- 🌈 **تم تاریک مدرن** - طراحی با رنگ‌های سیاه، نئونی و گرادیانت

### 📝 راهنمای نصب | Installation Guide

**🤖 اندروید:**
1. فایل `UltimateCalculator-X-android.apk` را دانلود کنید
2. در تنظیمات گوشی، گزینه **Install from Unknown Sources** را فعال کنید
3. روی فایل APK کلیک کرده و نصب کنید

**🪟 ویندوز (8.1 / 10 / 11):**
1. فایل `UltimateCalculator-X-windows.zip` را دانلود کنید
2. فایل ZIP را Extract کنید
3. فایل `advanced_calculator.exe` را اجرا کنید
4. اگر Windows SmartScreen هشدار داد، روی **More info** و سپس **Run anyway** کلیک کنید

## 🚀 توسعه محلی | Local Development

برای اجرای پروژه روی سیستم خودتان:

~~~bash
# Clone repository
git clone https://github.com/par123456/s.git
cd advanced-calculator

# Install dependencies
flutter pub get

# Run on connected device
flutter run

# Build APK
flutter build apk --release

# Build Windows EXE
flutter config --enable-windows-desktop
flutter build windows --release
~~~
